# Major RB Properties Website

Modern, responsive website for Major RB Properties, specializing in land and property development in Moshate Gardens, Soshanguve.

## Deployment Instructions

This is a static website. You can deploy it easily to any static hosting service:

### GitHub Pages
1. Push this repository to GitHub.
2. Go to **Settings > Pages**.
3. Select the `main` branch and `/root` directory.
4. Save and your site will be live at `https://[username].github.io/[repo-name]/`.

### Vercel / Netlify
1. Connect your GitHub repository.
2. The build command is not needed (it's plain HTML).
3. The publish directory is the root directory (`.`).

## Features
- Responsive design using Tailwind CSS.
- Smooth animations and transitions.
- Interactive property search (demo).
- Lead generation form (demo).
- WhatsApp integration for direct enquiries.

## File Structure
- `index.html`: Main landing page.
- `about.html`: About the company.
- `new-sale.html`: Featured property listing.
- `testimonials.html`: Client reviews.
- `404.html`: Custom error page.
- `script.js`: Interactive functionality.
- `styles.css`: Custom animation styles.
- `Major-RB-logo.png`: Company logo.
- `*.jpeg`: Property and background images.
